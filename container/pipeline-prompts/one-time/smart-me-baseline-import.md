# smart-me 认知基线导入

一次性任务：将 smart-me 项目中已识别的认知模式数据导入到 HappyClaw 的 cognitive-profile.md 中，作为认知观察系统的初始基线。

## 输入

读取以下文件：
1. smart-me 思维模式主文档：在当前工作区 `knowledge/personal-notes/thinking-and-learning-pattern.md`
2. 当前 cognitive-profile.md：在 `/workspace/global/cognitive/cognitive-profile.md`

## 导入内容

从 smart-me 文档的「一、跨情境稳定的核心模式」和「二、领域学习专项模式」提取 12 个模式。每个模式转换为 cognitive-profile.md 的稳定模式表格式：

| 维度 | 模式 | 首次 | 累计 | 状态 | 来源 |
|------|------|------|------|------|------|
| 决策 | 框架锁仓再展开细节 | smart-me | 3+ | 核心模式 | smart-me 基线 |

- 模式 1（框架锁仓）→ 决策
- 模式 2（注意力深度分配）→ 注意力分配
- 模式 3（选择性验证）→ 决策
- 模式 4（主动识别 gap）→ 思考路径
- 模式 5（AI 协作分权）→ 决策
- 模式 6（跨域映射）→ 思考路径
- 模式 7（理解驱动）→ 思考路径
- 模式 8（体验感知锚定）→ 思考路径
- 模式 9（短反馈循环）→ 决策

- 模式 11（功能边界优先语言边界）→ 思考路径
